Robot-Assisted Shoebox Tasks

Shoebox Tasks are visually structured educational activities widely used in the education of individuals with Autism Spectrum Disorder (ASD). Because the task goal can be understood directly from the visual arrangement of objects, they are well suited for computer vision and machine learning approaches.
Our research explores how visual information can be translated into symbolic task descriptions (PDDL), abstract action plans, and executable robot actions. We also develop a visual programming interface based on visual schedules, inspired by structured educational approaches such as TEACCH®, treating them as simple algorithms for robot programming.
The project connects three main components: an automated vision-based task solver, robots acting as teachers, and a visual programming interface that allows students to teach the robot how to solve tasks.
Our goal is to use familiar educational structures to foster interest in robotics among students with ASD and provide accessible pathways to computational thinking and robotics.
