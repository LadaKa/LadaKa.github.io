Vision-Based Solver

The Vision-Based Solver enables automatic analysis and execution of visually structured Shoebox Tasks. Using computer vision and machine learning, the system detects task components, interprets task goals, and generates action sequences for autonomous task completion.

Shoebox Tasks

Educational tasks designed for individuals with ASD, providing a structured environment for learning and assessment.

Task Solver

Vision-based analysis, task interpretation, planning, and automatic generation of task-solving actions.

Robotic Interface

Communication between the solver and robotic platforms, enabling autonomous task execution and demonstrations.
