Robotic Interface

The robotic interface allows the robot to execute actions generated by the task solver. The implementation of these actions depends on the robot type and the desired behavior (e.g., manipulation with objects, gestures, or voice commands).

We adapted Shoebox-tasks for various robotic platforms, currently focusing on the humanoid robot Nao while also experimenting with low-cost floor robots such as Ozobot.

The system supports two main use cases: the robot can act as a teacher, guiding the student through task execution, or the student can control the robot through the visual programming interface to perform the task.
